Sổ tay quản trị

Tự làm các việc thường gặp — không cần biết code.

Sửa giá / chữ trên trang
  1. Mở GitHub → bấm vào file cần sửa (ví dụ giá lò 85L: sp-tach-khoi-85l.html).
  2. Bấm biểu tượng bút chì ✏️ (Edit) góc trên phải.
  3. Dùng Ctrl+F tìm con số / câu chữ cần đổi (vd 16.500.000), sửa lại.
  4. Kéo xuống cuối → bấm Commit changes.
  5. Đợi ~1 phút, web tự cập nhật. Tải lại trang bằng Ctrl+Shift+R để thấy ngay (tránh cache).
Lưu ý: giá hiển thị ở nhiều nơi (thẻ sản phẩm trong san-pham.html và trang chi tiết sp-*.html) — đổi giá nhớ sửa cả 2 chỗ.

Web cập nhật / deploy như thế nào

Mỗi lần Commit trên GitHub → Cloudflare tự build & đăng sau ~1 phút. Không cần thao tác gì thêm. Không có trang admin kiểu wp-admin — chỉnh sửa qua GitHub là cách chính thức.

3 tài khoản / chìa khoá phải giữ
  • GitHub (github.com/duyoven) — chứa toàn bộ mã nguồn.
  • Cloudflare — hosting + tên miền + Worker. Có 2 Secret (ở Worker → Settings → Variables): ANTHROPIC_API_KEY (key AI) và STAFF_AUTH (user/mật khẩu nội bộ, dạng user:pass). Phải để kiểu Secret, không để Text.
  • Anthropic Console (console.anthropic.com) — tạo key + nạp credit cho AI chat & dịch.
Mất 1 trong 3 cái này là mất quyền quản trị phần tương ứng — lưu kỹ.
Đổi mật khẩu nội bộ / thêm nhân viên

Vào Cloudflare → Worker duyoven-vn-site → Settings → Variables → sửa Secret STAFF_AUTH. Định dạng: user1:matkhau1,user2:matkhau2. Lưu là áp dụng ngay cho cả Báo giá và Trang điều khiển này.

Gặp sự cố — xử lý nhanh
  • Sửa xong web chưa đổi: đợi 1–2 phút (đang build) + tải lại bằng Ctrl+Shift+R.
  • Chatbot / dịch không chạy: kiểm tra credit ở Anthropic Console (thường do hết tiền). Bấm "Kiểm tra ngay" ở trên để xác nhận.
  • Trang mất giao diện (trắng trơn): kiểm tra file style.css còn trên GitHub không.
  • Không vào được Báo giá/Trang này: kiểm tra Secret STAFF_AUTH trên Cloudflare còn không (mỗi lần build kiểu Text sẽ bị xoá — phải để Secret).
